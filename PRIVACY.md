# Privacy Policy for Gemini Tweak Extension

**Last Updated:** February 2026

## 1. Data Collection
The **Gemini Tweak Extension** does **not** collect, store, share, or transmit any personal information, browsing history, or user data to any external servers.

## 2. Local Storage
We use the Chrome `storage` API solely to save your customized settings locally on your device (e.g., preferred language, visibility options, and button states). These preferences never leave your browser.

## 3. Third-Party Services
This extension interacts exclusively with the `gemini.google.com` website to apply visual modifications. It does not communicate with any other third-party services or analytics providers.

## 4. Contact
If you have any questions about this policy, please contact the developer via the support section on the Chrome Web Store or the GitHub repository.
