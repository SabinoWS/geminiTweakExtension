# 🔧 Gemini Tweak Extension

Chrome 扩展程序，旨在通过隐私控制和生产力快捷方式自定义 **Google Gemini** 界面。

---

## ✨ 功能

### 🎛️ 快速模型选择器

![模型选择器](screenshots/buttonsPreview.png)

-用于在 Gemini 模型之间切换的快速访问按钮：
  - ⚡ **其实 (Fast)** - 更快的响应
  - 🧠 **推理 (Reasoning)** - 更适合复杂任务
  - 💎 **专业 (Pro)** - 最先进的模型

> 模型可以通过按钮直接访问（无需下拉菜单），并且上次使用的模型状态会自动保存并在返回页面时恢复。

### 🔒 隐私控制 (模糊/审查)

![隐私控制](screenshots/blurPreview.png)

使用模糊效果隐藏侧边栏部分，以获得更大的隐私：

| 部分 | 描述 |
|---------|-------------|
| **我的项目** | 隐藏最近项目部分 |
| **Gems** | 隐藏个性化 Gems 列表 |
| **对话历史** | 隐藏或限制可见对话 |

#### 历史记录模式：
- 👁️ **可见** - 显示所有对话
- 🔢 **受限** - 仅显示前 5 个对话（其余模糊）
- 🚫 **隐藏** - 对整个历史记录应用模糊

> **注意：** 我们使用模糊而不是 `display: none`，以保持 Gemini 的延迟加载正常工作。

### 🗑️ 快速删除对话

![快速删除](screenshots/fastDeletePreview.png)

- 在每个对话旁边添加一个 **垃圾桶按钮**。
- 允许快速删除对话而无需手动确认（流程已自动化）。
- **菜单始终可见**：垃圾桶和 3 点菜单都保持固定，方便访问。
- 图标颜色适用于浅色和深色主题。

## 🚀 安装

1. 克隆或下载此存储库
2. 打开 Chrome 并转到 `chrome://extensions/`
3. 启用 **开发者模式** (右上角)
4. 点击 **加载已解压的扩展程序**
5. 选择项目文件夹

## ⚙️ 设置

![设置弹出窗口](screenshots/modalPreview.png)

在弹出窗口的顶部，您可以找到 **快速操作** 按钮：
- **默认**: 恢复原始扩展设置。
- **全部关闭**: 禁用所有功能。
- **全部开启**: 启用所有功能。

下面是详细选项：

| 类别 | 选项 | 描述 |
|-----------|-------|-----------|
| **常规** | 替代按钮 | 在顶部启用快速模型切换按钮 |
| | 启动时关闭侧边栏 | 页面加载时自动关闭侧边栏 |
| **简洁界面** | 隐藏问候 | 移除 "你好，用户" 消息 |
| | 隐藏欢迎 | 移除初始欢迎文本 |
| | 隐藏建议 | 移除提示建议卡片 |
| **可见性**| 隐藏历史记录 | 应用模糊/隐藏最近对话列表 |
| | 限制历史记录 (5) | 仅显示最近的 5 个项目 |
| | 隐藏我的项目 | 隐藏 "我的项目" 部分 |
| | 隐藏 Gems | 隐藏 "Gems" 部分 |
| **生产力**| 快速删除 | 添加垃圾桶图标以无需确认即可删除 |
| | 始终显示菜单 | 保持菜单（3 点）始终可见，无需悬停 |

## 🎨 技术

- **Manifest V3** - 最新的 Chrome 扩展格式
- **Vanilla JavaScript** - 无外部依赖
- **纯 CSS** - 模糊效果和过渡
